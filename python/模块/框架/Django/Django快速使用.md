## 安装Django
```sh
pip install diango
```
## 创建项目
切换到项目目录,diango-admin startproject mydemo
```sh
cd /Users/mt/Documents/pyproject/MyDjango

/Users/mt/.local/bin/django-admin startproject mydemo
```

## 默认项目文件的作用
```
mydemo
├── manage.py
└── mydemo
    ├── __init__.py
    ├── asgi.py
    ├── settings.py
    ├── urls.py
    └── wsgi.py
  
```
